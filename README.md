# libParade
A generic download accelerator/manager library.

Currently in an unusable state.

- [ ] YouTube Downloader

  - [ ] Playlist
  - [ ] Channel
- [ ] Spotify Downloader
  - [ ] Track
  - [ ] Album
  - [ ] Playlist
  - [ ] Artist
- [ ] Plugin System

## Dependencies

- Aria2 (Optional)
- FFMPEG (Optional)
